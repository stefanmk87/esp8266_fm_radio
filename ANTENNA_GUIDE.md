## üö® TROUBLESHOOTING AUDIO PROBLEMS

### **Problem: Audio Breaking Up**

**Causes:**
1. RSSI < 35 (weak signal) ‚Üí **Add external antenna!**
2. Wrong de-emphasis (75¬µs vs 50¬µs) ‚Üí **Fixed in new firmware**
3. Soft-mute not enabled ‚Üí **Fixed in new firmware**
4. Antenna too close to ESP-01 WiFi ‚Üí **Move apart 10cm+**

**Solutions:**
- Upload latest firmware (includes audio fixes)
- Add 75cm wire antenna
- Position antenna away from ESP-01
- Try different FM stations (stronger signal)

---

### **Problem: Noise/Static**

**Causes:**
1. Power supply noise ‚Üí **Add more capacitors**
2. WiFi interference ‚Üí **Position antenna away**
3. Weak signal + no soft-mute ‚Üí **Upload new firmware**

**Solutions:**
- Add 100¬µF capacitor close to Si4703 VCC
- Keep antenna >10cm from ESP-01
- Enable soft-mute (new firmware has this)

---

### **Problem: Mono Audio (Not Stereo)**

**Causes:**
1. setMono(false) not called ‚Üí **Fixed in firmware**
2. Weak signal auto-switches to mono ‚Üí **Improve antenna**

**Solutions:**
- Upload latest firmware
- Check web interface shows "STEREO"
- If shows "MONO", signal too weak ‚Üí add antenna
